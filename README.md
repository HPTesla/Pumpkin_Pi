Pumpkin Pi
==========

Illuminate a Jack-o'-Lantern using LEDs and Raspberry Pi.

Usage
-----

**sudo ./pumpkinPi.py hh:mm**

hh:mm is the duration in hours and minutes to run.

This program must be run as root in order to access the GPIO pins.
